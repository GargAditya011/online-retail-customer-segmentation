## Project Overview
This project focuses on analyzing online retail transaction data to uncover customer purchasing behavior, identify valuable customer segments, and generate actionable business insights using data analytics and machine learning techniques.

The project includes:
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Customer Purchase Behavior Analysis
- Customer Segmentation using K-Means Clustering
- Business Insights & Visualization

---

## Problem Statement
Online retail businesses generate large volumes of transactional data, making it difficult to identify customer behavior patterns and high-value customer groups.

The objective of this project is to:
- Analyze customer purchasing trends
- Identify valuable customer segments
- Improve customer targeting strategies
- Generate business insights from retail transaction data
